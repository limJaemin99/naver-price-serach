네이버 쇼핑 검색 API 테스트

Client ID / Secret 을 넣고 상품명을 검색하면 상품명·최저가·판매처·링크가 표시됩니다.

⚠️ 이 페이지를 파일로 직접 열었습니다(file://). 네이버 API는 브라우저 직접 호출을 CORS 정책으로 막습니다. 같은 폴더의 server.js 를 실행한 뒤 http://localhost:3000 으로 접속해야 결과가 옵니다.
실행 방법: 이 폴더에서 터미널 열고 node server.js → 브라우저에서 http://localhost:3000
키는 이 브라우저(localStorage)에만 저장되며 외부로 전송되지 않습니다.
